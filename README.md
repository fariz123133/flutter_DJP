# M-Pajak Dashboard Clone (Flutter)

Proyek ini adalah implementasi ulang (clone) dari antarmuka pengguna (UI) aplikasi **M-Pajak (DJP)** menggunakan **Flutter**. Fokus utama proyek ini adalah meniru desain Dashboard yang modern, bersih, dan interaktif.

## ğŸš€ Cara Menjalankan (Getting Started)

Ikuti langkah-langkah berikut untuk menjalankan proyek ini di komputer Anda.

### Prasyarat
Pastikan Anda sudah menginstal:
* [Flutter SDK](https://docs.flutter.dev/get-started/install)
* VS Code atau Android Studio

### Instalasi

1.  **Clone Repository**
    Buka terminal dan jalankan perintah berikut:
    ```bash
    git clone [https://github.com/fariz123133/flutter_DJP.git](https://github.com/fariz123133/flutter_DJP.git)
    ```

2.  **Masuk ke Direktori Project**
    ```bash
    cd flutter_DJP
    ```

3.  **Install Dependencies**
    Unduh semua paket yang dibutuhkan:
    ```bash
    flutter pub get
    ```

4.  **Jalankan Aplikasi**
    Pastikan Emulator atau Device fisik sudah terhubung, lalu jalankan:
    ```bash
    flutter run
    ```
